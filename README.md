# Useful Scripts

A collection of small Python automation scripts.

## Scripts

### 1. backup.py
Creates a timestamped backup of a folder.

`python backup.py`

### 2. rename_files.py
Adds a prefix to all filenames in a folder.

`python rename_files.py`

### 3. download_images.py
Downloads images from URLs listed in a text file.

`python download_images.py`

## Requirements

- Python 3.12
- requests (see `requirements.txt`)
